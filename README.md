# SecurityBox - Siber Güvenlik Analiz Aracı #

SecurityBox, phishing e-postaları, şüpheli bağlantıları ve zayıf şifreleri tespit etmek için geliştirilmiş web tabanlı temel seviye geliştirmeye açık bir siber güvenlik analiz aracıdır.
Kullanımı gayet basit bir arayüz içerir. Sosyal mühendislik gibi tehditlere karşı temel bir savunma mekanizması içerir.
## 🎯 Özellikleri -->  Mail analizi, URL kontrolü, Kullanıcı dostu şifre oluşturma(Sosyal medya, mobil bankacılık, web uygulamaları vs. için).

## Backend side
- **Python 3.x**
- **FastAPI**: Hızlı ve modern web framework
- **Pydantic**: Veri doğrulama

## Frontend side

- **HTML5**
- **JavaScript (ES6+)**
- **Tailwind CSS**: CSS framework
- **Font Awesome**: İkon kütüphanesi

**Projeyi klonlamak**
```bash
git clone https://github.com/kullanıcı-adi/projeSiberWEb.git
cd projeSiberWEb
```

**Gerekli paketleri yükleyin**
```bash
pip install fastapi uvicorn python-multipart
```

4. **Uygulamayı başlatmak için**

```bash
uvicorn API:app --reload
```

✅ **Verileriniz kaydedilmez!** Tüm analizler istemci tarafında gerçekleştirilir ve hiçbir veri sunucuda depolanmaz.
